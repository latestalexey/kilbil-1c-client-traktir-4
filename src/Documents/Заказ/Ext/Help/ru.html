Документ "Заказ".

Менеджер ресторана на своем рабочем месте может только просматривать оформленные заказы. Никаких изменений в состав реквизитов заказа с рабочего места менеджера вносить нельзя.

Функция просмотра журнала вызывается из меню «Документы», пункт «Заказы».

Заказы, по которым предоставлялись скидки или наценки, отмечаются в журнале заказов символами "Стрелочка вверх" и "Стрелочка вниз", соответственно.

Заказы, по которым производились удаления, отмечены символом "Х" в колонке «Статус».

 Для просмотра всей информации по заказу используется форма документа «Заказ» (открывается двойным щелчком мыши по строке заказа). Эта форма доступна только для просмотра.

В форме, на нескольких закладках, можно увидеть следующую информацию по заказу:

·       номер, дата и время открытия и закрытия заказа;

·       информация по заказу (место реализации, посадочное место, количество посетителей, официант, кассир, клиент).

·       блюда (товары), их количество, цена и сумма, скидки и наценки по каждой позиции;

Закладка «Скидки» появляется в том случае, если в заказе применялись скидки (наценки).

На закладке указывается тип для каждой скидки (наценки), ее размер и сумма, и общая сумма скидок (наценок).

На закладке «Протокол расчетов» представлены данные по каждому из вариантов оплаты, которые применялись при оплате данного заказа.
