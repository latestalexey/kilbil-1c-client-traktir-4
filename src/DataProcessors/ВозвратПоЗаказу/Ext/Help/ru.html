Обработка "Возврат по заказу".

В ресторане могут возникнуть ситуации, когда необходимо произвести полный либо частичный возврат денег посетителю по уже оплаченному заказу. Причинами для такого удаления могут быть, например, рекламация, либо невозможность приготовить заказанное и оплаченное блюдо в ресторане быстрого обслуживания, ошибка официанта, и т.п.

Система позволяет произвести возврат по заказу полностью (с возвратом всей уплаченной суммы), либо возврат по отдельным позициям заказа,  а также возврат за счет уменьшения количества по отдельным позициям.

Вызывается из журнала заказов.

Кнопка "Вернуть все", используется, если необходимо произвести полный возврат.

Если необходимо произвести возврат по отдельным позициям, необходимо поочередно установить курсор на выбранные позиции и задать возвращаемое количество кнопкой «Кол-во». Возвращаемое количество не может быть больше, чем указано в заказе. При этом, выбранное количество по данной позиции отобразится в графе «Вернуть», а в графе «Сумма» автоматически отобразится сумма возврата.

Выбор варианта возврата денег из списка доступных вариантов происходит по соответствующей кнопке.

Выбор ККМ из списка доступных вариантов происходит по соответствующей кнопке.

Кнопка "Оплата" служит для расчета с посетителем.
